# GroneStats
Todo lo que haré para GroneStats o GroneStatistics, el canal de Youtube y X que se dedica al análisis de datos del club Alianza Lima.
Librerias utilizadas: ScraperFC de oseymur https://github.com/oseymour/ScraperFC/ 
Se le incluye en el código la manera de obtener los datos de la Liga 1 Max.
